# Il Gioco della Papera

Gioco multiplayer a turni ispirato al classico Gioco dell'Oca, in versione "papera".

Ogni giocatore tira il dado, avanza sul tabellone e risponde a una domanda.
Ci sono caselle speciali che possono far avanzare, arretrare o rilanciare.
Vince chi raggiunge per primo la casella finale.

Il gioco usa Firebase per autenticazione, salvataggio della partita in tempo reale e notifiche email del turno.
